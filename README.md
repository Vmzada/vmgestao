
# 🚀 VM Gestão — Financeiro e Controle de Estoque

Sistema profissional de gestão financeira, controle de estoque e PDV (Frente de Caixa). Desenvolvido para oferecer controle total, agilidade e segurança para o seu negócio.

## 📤 Como subir o projeto para um NOVO repositório (Limpo e Profissional)

Se você quer criar um novo repositório e garantir que as conversas com a IA não apareçam no histórico, siga estes passos no seu terminal:

1. **Apague o histórico Git local atual:**
   ```bash
   rm -rf .git
   ```

2. **Inicie um novo repositório do zero:**
   ```bash
   git init
   ```

3. **Adicione TODOS os arquivos (O ponto final é obrigatório):**
   ```bash
   git add .
   ```

4. **Crie o primeiro e único commit oficial:**
   ```bash
   git commit -m "🚀 VM Gestão - Versão Oficial 1.0"
   ```

5. **Conecte ao seu novo repositório (Crie um repositório vazio no GitHub primeiro):**
   ```bash
   git remote add origin https://github.com/SEU_USUARIO/NOVO_REPOSITORIO.git
   ```

6. **Envie tudo para a nuvem:**
   ```bash
   git push -u origin main --force
   ```

## ✨ Funcionalidades

- **💰 Frente de Caixa (PDV)**: Vendas rápidas com seleção de método de pagamento.
- **📦 Gestão de Estoque**: Cadastro completo com alertas de nível crítico.
- **📊 Histórico de Vendas**: Edite valores, mude formas de pagamento ou exclua vendas com estorno automático de estoque.
- **📊 Dashboard**: Faturamento diário, ticket médio e gráficos.
- **🛡️ Auditoria**: Registro de todas as ações importantes.
- **🔒 Segurança**: Controle de acesso por login e senha.
- **🌗 Dark Mode**: Interface otimizada para qualquer ambiente.
